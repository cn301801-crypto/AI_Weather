📘 About This Project

🌦 Project Overview

This project presents a Multi-Model Hybrid Weather Prediction System that combines both image-based analysis and numerical weather data to improve prediction accuracy.

🧠 Machine Learning Approach

The system integrates multiple learning paradigms:

  • Supervised Learning: Used for training predictive models like CNN and XGBoost using labeled datasets.
  • Unsupervised Learning: Applied for pattern discovery and clustering in weather data to enhance understanding of hidden structures.
  • Hybrid Model: Combines outputs from different models to generate a final prediction.

🖼 Image-Based Prediction

A Convolutional Neural Network (CNN) is used to analyze weather images (such as cloudy, sunny, rainy conditions) and extract visual patterns.

📊 Numerical Data Prediction

Weather parameters such as:

  • Temperature
  • Humidity
  • Pressure
  • Wind Speed

are processed using XGBoost, a powerful supervised learning algorithm.

🔗 Hybrid Decision System

The final prediction is generated by combining both:

  • Image-based prediction
  • Numerical data prediction

This hybrid approach improves robustness and reduces prediction errors.

⚙️ Data Processing Pipeline

  • Data Cleaning & Preprocessing
  • Feature Scaling using Standardization
  • Image Resizing and Normalization
  • Model Training (Supervised + Unsupervised)
  • Model Integration

🎯 Objective

To build a reliable and intelligent weather prediction system that leverages both visual and numerical data using advanced machine learning techniques.
